This simualtion works in ROS2 Humble and Gazebo Harmonic. The system was built in Ubuntu 22.04 distro for Linux. all the necessary packages should be downloaded, most importantly, ros_gz_bridge. 

This system works in principle with quarter-car suspension system utulizing full state feedback vectors (k) and N vectors to simulate control on the suspension system. 
This robot lacks an IMU sensor, so the state of y_c and y_k are approximated to be the same. This state mirroring is done to solely simulate the human-robot interaction.
